Política de Privacidade — NutriCare

Última atualização: 13 de maio de 2026

O NutriCare valoriza sua privacidade. Esta política explica quais dados coletamos, como os utilizamos e seus direitos como usuário.

1. Dados que coletamos

Para gerar planos nutricionais personalizados, coletamos as seguintes informações fornecidas por você:

  • Perfil de saúde: peso, altura, idade, sexo, nível de atividade física, objetivo (emagrecer, ganhar massa, manter peso) e condições de saúde autorrelatadas (comorbidades).
  • Histórico de consultas: perguntas e respostas geradas dentro do app.
  • Progresso: registros de peso e medidas ao longo do tempo.

2. Dados de pagamento

O processamento de pagamentos do plano Premium Trimestral é feito exclusivamente pelo Stripe. O NutriCare não armazena dados de cartão de crédito. O Stripe coleta e processa essas informações conforme a Política de Privacidade do Stripe.

3. Onde os dados ficam armazenados

Os dados do seu perfil, histórico e progresso são armazenados:

  • No seu dispositivo: via localStorage do navegador (com criptografia AES-256-GCM para dados sensíveis).
  • No servidor: banco de dados Supabase (PostgreSQL) para consultas vinculadas a contas autenticadas.
  • Se você não criar conta, seus dados ficam apenas no seu dispositivo e podem ser perdidos ao limpar o cache do navegador.

4. Como usamos seus dados

Seus dados são utilizados exclusivamente para:

  • Gerar planos alimentares e recomendações nutricionais personalizadas
  • Exibir seu histórico de progresso
  • Melhorar o serviço (estatísticas anônimas e agregadas)

Não vendemos, compartilhamos ou comercializamos seus dados pessoais com terceiros.

5. PIN Profissional

Nutricionistas parceiros podem utilizar um PIN para liberar o Premium aos seus clientes. O PIN é armazenado com hash criptográfico (SHA-256) — nunca em texto puro.

6. Seus direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Acessar seus dados armazenados
  • Solicitar a exclusão completa dos seus dados
  • Revogar consentimento a qualquer momento

Para exercer esses direitos, entre em contato pelo e-mail abaixo.

7. Segurança

Utilizamos:

  • Criptografia AES-256-GCM para dados sensíveis no dispositivo
  • Conexão HTTPS em todas as comunicações com o servidor
  • Autenticação JWT para acesso à API

8. Crianças

O NutriCare não é direcionado a menores de 13 anos. Se você é pai/mãe e acredita que seu filho forneceu dados sem consentimento, entre em contato para remoção imediata.
